FuncKeys application set

FuncKeys, FuncKeysCPL and FuncKeysUnReg builds a set of applications to unregister function keys that have been registered as hotkeys by windows mobile or a windows mobile application.
FuncKeys
The application to manage which keys will be unregistered
FuncKeysCPL
A control panel applet which simply starts the FuncKeys application
FuncKeysUnReg
The application that unregisters the function keys on windows startup

Background

In example, the phone application - cprog.exe - registers the hotkeys F3 and F4 on startup. The 'function keys' are also mapped to other named keys, see 'Mapping from winuserm.h'. If you select F3 and F4 in FuncKeys, these keys will be unregistered on startup by FuncKeysUnReg.

FuncKeysCPL

This little app enables you to launch FuncKeys from Start-Settings-System. It is a simple control panel applet that will launch FuncKeys.

FuncKeys

This is the application to manage the keys that will be unregistered by FuncKeysUnReg . FuncKeys is simply an interface to the registry key HKLM\Software\Intermec\FuncKeys:KeyList. This registry key is used by FuncKeysUnReg to read the keys to unregister these keys on startup.

FuncKeysUnReg

This application will use UnregisterFunc to deregister system wide Function Key assignments. Some applications will register Function Keys during there start (i.e. the phone app, cprog.exe). If you use the autostart component (FuncKeysUnreg), these app's function keys will be deregistered. To get there default function key assignment back, the apps have to be relaunched. You can remove FuncKeysUnReg from the startup folder and warmboot the device or deselect  F3 and F4 in FuncKeys and then reboot.

Installation

You can use the installation cab file or install the tool set manually.
